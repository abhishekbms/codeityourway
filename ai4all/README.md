# AI 4 All — Day 1: Agent Flows in Microsoft Copilot Studio
## PPTX Generator

This script generates the full PowerPoint presentation for the AI 4 All Day 1 session on Agent Flows in Microsoft Copilot Studio.

### Prerequisites
- Python 3.8+
- pip

### Setup & Run

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Generate the presentation:
   ```
   python generate_presentation.py
   ```

3. Open the output file:
   ```
   AI4All_Day1_AgentFlows.pptx
   ```

### Output
The script creates `AI4All_Day1_AgentFlows.pptx` in the current directory with 14 fully formatted slides.
